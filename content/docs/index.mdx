# 为什么你需要彻底掌握 SVG？

在当今的前端开发中，HTML 负责结构，CSS 负责样式，JavaScript 负责逻辑。而当你需要绘制**图表、动态图标、复杂背景、甚至是在线白板应用**时，你需要一个专用的渲染引擎。

HTML 和 CSS 无法画出平滑的贝塞尔曲线；Canvas 是一张死板的像素位图，难以响应事件；WebGL 又过于底层和复杂。
**SVG（可缩放矢量图形）**，作为一种基于 XML 的原生 DOM 标签，是唯一能够完美平衡**矢量高清、可编程性、事件交互与样式定制**的技术。

本指南分为 23 个核心章节，带你从最基础的 `<circle>` 画起，一路深入到矩阵变换、高级滤镜、React 组件化封装，甚至涉及生成艺术与安全防御。
