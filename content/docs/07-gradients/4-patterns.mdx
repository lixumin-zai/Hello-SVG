# 7.4 图案填充 (`<pattern>`)

图案填充允许我们将一个 SVG 图形（或位图）定义为“瓷砖”，然后无限平铺来填充另一个图形。

## 7.4.1 定义图案

使用 `<pattern>` 元素定义图案单元。

```xml
<defs>
  <pattern id="dots" x="0" y="0" width="20" height="20" patternUnits="userSpaceOnUse">
    <circle cx="10" cy="10" r="5" fill="orange" />
  </pattern>
</defs>

<rect fill="url(#dots)" ... />
```

## 7.4.2 关键属性

- **`width`, `height`**: 定义图案单元（瓷砖）的大小。
- **`patternUnits`**:
    - `userSpaceOnUse` (常用): `width/height` 是绝对像素值。图案固定在坐标系中，不随被填充物体缩放。
    - `objectBoundingBox` (默认): `width/height` 是百分比（0-1）。图案会随物体缩放。

## 7.4.3 嵌套使用

图案内部可以是任何 SVG 元素，甚至可以包含另一个引用了图案的形状！这使得创建分形图案成为可能。

## 7.4.4 应用场景

1.  **背景纹理**: 网格纸、斜线纹理、波点。
2.  **数据可视化**: 在黑白打印或色盲友好的图表中，用不同的纹理区分数据系列（如柱状图）。
