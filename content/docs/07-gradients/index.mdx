# Part 7: 渐变与图案

如果说 `<path>` 给了 SVG 骨架，那么**渐变 (Gradients)** 和 **图案 (Patterns)** 则给了它丰富的血肉。

单一的纯色填充（Flat Color）有时显得过于单调。SVG 提供了强大的着色服务器（Paint Server）机制，允许我们定义复杂的着色规则，然后将其应用到 `fill` 或 `stroke` 属性上。

在本章中，我们将学习：

1.  **线性渐变 (`<linearGradient>`)**：颜色沿直线变化。
2.  **径向渐变 (`<radialGradient>`)**：颜色从一点向外辐射。
3.  **网格渐变 (Mesh Gradient)**：SVG 2.0 的新特性，虽然浏览器支持尚不完善，但值得了解。
4.  **图案 (`<pattern>`)**：使用平铺的图形作为填充。

这些特性不仅可以用于填充图形内部，也可以用于描边，甚至可以作为蒙版的一部分。
