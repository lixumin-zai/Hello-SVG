# 7.1 线性渐变 (`<linearGradient>`)

线性渐变是颜色沿一条直线（称为**渐变向量**）平滑过渡的效果。

## 7.1.1 定义与引用

在 SVG 中，渐变通常定义在 `<defs>` 元素中，然后通过 `id` 被引用。

```xml
<svg>
  <defs>
    <linearGradient id="myGradient" x1="0%" y1="0%" x2="100%" y2="0%">
      <stop offset="0%" stop-color="blue" />
      <stop offset="100%" stop-color="red" />
    </linearGradient>
  </defs>
  
  <rect fill="url(#myGradient)" ... />
</svg>
```

## 7.1.2 渐变向量 (`x1`, `y1`, `x2`, `y2`)

这四个属性定义了渐变的起点和终点。
- **默认值**: `x1="0%"` `y1="0%"` `x2="100%"` `y2="0%"` (从左到右)。
- **单位**: 可以是百分比（相对于图形的包围盒），也可以是用户坐标（如果 `gradientUnits="userSpaceOnUse"`）。

### 常见方向
- **从上到下**: `x1="0%" y1="0%" x2="0%" y2="100%"`
- **45度角**: `x1="0%" y1="0%" x2="100%" y2="100%"`

## 7.1.3 色标 (`<stop>`)

渐变由多个色标组成。每个 `<stop>` 元素有两个关键属性：
- **`offset`**: 位置（0% 到 100%）。
- **`stop-color`**: 颜色。
- **`stop-opacity`**: 透明度。
