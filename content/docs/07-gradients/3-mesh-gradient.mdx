# 7.3 网格渐变 (Mesh Gradient)

**注意**: 网格渐变是 SVG 2.0 规范的一部分，目前（截至 2024 年）浏览器支持度仍然非常有限。本节主要介绍概念。

## 7.3.1 什么是网格渐变？

线性渐变和径向渐变都很规则。如果你想创建如丝绸般流动、或者复杂曲面的光影效果，传统的渐变很难做到。

网格渐变允许你定义一个二维的网格（Coons Patch），你可以控制网格中每个顶点的颜色和坐标。颜色会在网格内部进行双线性插值或双三次插值。

## 7.3.2 语法预览

```xml
<meshGradient x="50" y="50">
  <meshRow>
    <meshPatch>
      <stop stop-color="red" />
      <stop stop-color="blue" />
      <stop stop-color="green" />
      <stop stop-color="yellow" />
    </meshPatch>
  </meshRow>
</meshGradient>
```

## 7.3.3 替代方案

由于浏览器支持不佳，目前要在 Web 上实现类似效果，通常使用：
1.  **多个径向渐变叠加**: 使用高斯模糊和混合模式。
2.  **Canvas / WebGL**: 编写 Shader 实现。
3.  **SVG 滤镜**: 使用 `<feTurbulence>` 和 `<feDisplacementMap>` 模拟流体色彩。
