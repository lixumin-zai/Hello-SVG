# 7.2 径向渐变 (`<radialGradient>`)

径向渐变是从一个焦点向外扩散的颜色过渡，通常用于模拟光源或球体效果。

## 7.2.1 关键属性

径向渐变比线性渐变多几个参数：

- **`cx`, `cy`**: 终点圆（最大圆）的圆心。默认 `50%`。
- **`r`**: 终点圆的半径。默认 `50%`。
- **`fx`, `fy`**: 焦点（Focal Point），即颜色渐变的起始点。默认等于 `cx`, `cy`。

## 7.2.2 焦点偏移 (`fx`, `fy`)

最有趣的效果来自于当 `fx, fy` 不等于 `cx, cy` 时。这会让高光点偏离几何中心，产生强烈的 3D 球体质感。

```xml
<radialGradient id="sphereGrad" cx="50%" cy="50%" r="50%" fx="25%" fy="25%">
  <stop offset="0%" stop-color="white" />
  <stop offset="100%" stop-color="blue" />
</radialGradient>
```
上面的代码会把白色的高光点移到左上角，看起来像光从左上方打过来。

## 7.2.3 铺展方法 (`spreadMethod`)

当渐变范围没有覆盖整个图形时，剩下的区域怎么办？`spreadMethod` 属性决定了这一点：

- **`pad`** (默认): 使用起始或结束颜色填充剩余区域。
- **`reflect`**: 镜像反射渐变。
- **`repeat`**: 重复渐变。
