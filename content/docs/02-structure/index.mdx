SVG 不仅仅是画图，它是一种文档格式。要驾驭 SVG，必须理解它的骨架——文档结构。

## 本章概览

1.  **SVG 文件剖析**：理解 `<svg>` 标签的核心属性，特别是让人头大的 `viewBox`。
2.  **坐标系统**：为什么 SVG 里的 (0,0) 在左上角？如何建立自己的坐标系？
3.  **嵌入方式**：`<img>`, `<object>`, `<iframe>` 还是内联？不同的嵌入方式决定了你能做什么。
4.  **命名空间**：为什么有时候 SVG 代码复制进去却显示不出来？

本章最核心的内容是 **ViewBox**。一旦你理解了它，你就掌握了 SVG 缩放的钥匙。
