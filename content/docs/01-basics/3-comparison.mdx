# 1.3 SVG 与其他图形技术对比

在现代 Web 开发中，我们有多种绘制图形的选择。了解每种技术的优缺点，才能在项目中做出正确的架构决策。

## 1.3.1 SVG vs. Canvas

这是最常见的对比。两者都可以绘制 2D 图形，但工作原理截然不同。

| 特性 | SVG (Scalable Vector Graphics) | Canvas (HTML5 `<canvas>`) |
| :--- | :--- | :--- |
| **本质** | **矢量 (Vector)** | **栅格 (Raster)** |
| **编程模型** | **声明式 (Declarative)**: "这里有一个圆" | **命令式 (Imperative)**: "把画笔移到 (x,y)，画一个圆" |
| **DOM** | 每个图形都是独立的 DOM 元素 | 只是一个像素画布，内部没有 DOM |
| **事件处理** | 支持事件监听 (click, hover) | 需要手动计算坐标来实现交互 |
| **CSS 支持** | 支持 CSS 样式和动画 | 不支持 CSS (只能用 JS 绘制) |
| **性能** | **对象数量少时高性能** (DOM 操作开销大) | **对象数量多时高性能** (直接操作像素) |
| **适用场景** | 图表、地图、UI、高保真矢量图 | 游戏、粒子效果、图像处理、超大数据量可视化 |

**结论**: 如果你需要交互性强、可访问性好、且图形数量在数千以内，选 SVG。如果你要做游戏或者渲染十万个粒子，选 Canvas。

## 1.3.2 SVG vs. WebGL

WebGL (Web Graphics Library) 是基于 OpenGL ES 的 JavaScript API，主要用于 3D 图形，但也可以用于高性能 2D。

- **维度**: WebGL 主打 3D，SVG 专注于 2D。
- **复杂度**: WebGL 学习曲线极陡峭（Shader, 矩阵变换），SVG 相对简单直观。
- **硬件加速**: WebGL 完全依赖 GPU，性能极强。SVG 主要依赖 CPU (部分浏览器有 GPU 加速)，但在处理复杂滤镜时可能卡顿。

**结论**: 除非你需要 3D 效果或者极端的渲染性能，否则不要轻易使用 WebGL 来替代 SVG。

## 1.3.3 SVG vs. CSS Shapes & Clip-path

CSS 也可以绘制简单的形状（如圆角、三角形）。

- **CSS Shapes**: 通过 `border-radius`, `transform: rotate` 等技巧可以画出圆形、三角形、梯形。
- **CSS Clip-path**: 可以裁剪出多边形。

**对比**:
- **CSS**: 适合简单的装饰性几何图形，代码量极少，且不增加额外的 DOM 节点（伪元素）。
- **SVG**: 适合任何复杂的形状，特别是曲线、不规则路径。

**结论**: 如果只是画个圆或者三角形箭头，CSS 就够了。如果是复杂的图标或插画，请用 SVG。

## 1.3.4 SVG vs. PNG / JPEG / WebP

这是矢量与栅格的再次对决，但在静态资源领域。

| 格式 | 最佳用途 | 缺点 |
| :--- | :--- | :--- |
| **SVG** | 图标、Logo、插画、简单图表 | 不适合照片，复杂纹理会导致文件过大 |
| **JPEG** | **照片**、色彩丰富的写实图片 | 有损压缩，不支持透明，文字边缘会有噪点 |
| **PNG** | 需要透明背景的**非矢量图** | 文件体积通常较大 |
| **WebP** | 通用现代格式 (替代 JPEG/PNG) | 兼容性已不是大问题，但仍是栅格图 |

**结论**:
- 是照片吗？ -> JPEG / WebP
- 是图标/Logo/插画吗？ -> **SVG**
- 是截图吗？ -> PNG / WebP

## 1.3.5 SVG vs. PDF

PDF (Portable Document Format) 和 SVG 有共同的祖先 (PostScript)，但用途完全不同。

- **PDF**: 这里的“P”代表 **Portable (便携)**。它的目标是**打印**，确保在任何设备上看到的文档版式完全一致（分页、字体嵌入）。PDF 是一个最终交付格式，不是 Web 渲染格式。
- **SVG**: 这里的“S”代表 **Scalable (可缩放)**。它的目标是**屏幕显示**，特别是响应式 Web 设计。

**结论**: 如果你要让用户下载合同或发票打印，用 PDF。如果你要在网页上展示图表或图标，用 SVG。
