# 4.4 弧线命令 (`A`, `a`)

弧线命令 `A` 是 SVG 中参数最多、最难掌握的命令。它用于创建椭圆或圆的一部分。

## 4.4.1 命令格式

`A rx ry x-axis-rotation large-arc-flag sweep-flag x y`

一共 7 个参数：

1.  **rx**: 椭圆 x 轴半径。
2.  **ry**: 椭圆 y 轴半径。
3.  **x-axis-rotation**: 椭圆相对于 x 轴的旋转角度（顺时针，度数）。
4.  **large-arc-flag**: 大弧标志（0 或 1）。决定选择大弧还是小弧。
5.  **sweep-flag**: 扫描方向标志（0 或 1）。决定顺时针还是逆时针绘制。
6.  **x**: 终点 x 坐标。
7.  **y**: 终点 y 坐标。

## 4.4.2 参数详解

### 半径 (`rx`, `ry`)
这两个参数定义了潜在的椭圆的大小。如果 `rx = ry`，则画的是圆弧。

### 旋转 (`x-axis-rotation`)
定义椭圆本身的旋转角度。对于圆弧（rx=ry），这个参数没有视觉效果。

### 两个标志位 (`large-arc-flag`, `sweep-flag`)
给定起点、终点和半径，通常有 4 种可能的弧线连接这两点。这两个标志位就是用来从这 4 种可能性中选择一种。

-   **large-arc-flag**:
    -   `0`: 小弧（小于 180 度）。
    -   `1`: 大弧（大于 180 度）。
-   **sweep-flag**:
    -   `0`: 逆时针方向（Counter-Clockwise）。
    -   `1`: 顺时针方向（Clockwise）。

### 交互演示：理解弧线

在下面的演示中，我们尝试绘制不同的弧线。

<div className="my-8">
  <PathDemo initialD="M 100 100 A 50 50 0 1 1 150 150" />
</div>

这段代码 `M 100 100 A 50 50 0 1 1 150 150` 的意思是：
- 起点 (100, 100)
- 半径 50, 50 (圆)
- 旋转 0
- 大弧标志 1 (选择大弧)
- 顺时针标志 1 (顺时针)
- 终点 (150, 150)

### 常见用例：画饼图

弧线命令最常见的用途之一是绘制饼图或圆环进度条。

```xml
<!-- 一个简单的 3/4 圆 -->
<path d="M 100 100 L 100 20 A 80 80 0 1 1 180 100 Z" fill="orange"/>
```
1. `M 100 100`: 移动到圆心。
2. `L 100 20`: 画线到正上方 (12点方向)。
3. `A 80 80 0 1 1 180 100`: 画一个大弧到右侧 (3点方向)。
4. `Z`: 闭合回到圆心。
