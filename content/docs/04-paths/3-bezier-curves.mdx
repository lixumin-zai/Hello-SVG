# 4.3 贝塞尔曲线

贝塞尔曲线（Bézier Curve）是计算机图形学中最重要的曲线。在 SVG 中，我们主要使用两种贝塞尔曲线：**三次贝塞尔曲线 (Cubic)** 和 **二次贝塞尔曲线 (Quadratic)**。

## 4.3.1 三次贝塞尔曲线 (`C`, `c`, `S`, `s`)

三次贝塞尔曲线需要三个点：**两个控制点**和一个**终点**。起点由上一个命令的终点决定。

- **格式**: `C x1 y1, x2 y2, x y`
- **参数**:
    - `(x1, y1)`: 起点的控制点。
    - `(x2, y2)`: 终点的控制点。
    - `(x, y)`: 曲线的终点。

控制点就像磁铁一样吸引曲线，但曲线通常不经过控制点。

### 交互演示：C 命令

下面的路径绘制了一条波浪线。试着调整控制点坐标，观察曲线形状的变化。

<div className="my-8">
  <PathDemo initialD="M 10 100 C 50 0, 150 200, 200 100" />
</div>

- 起点: (10, 100)
- 控制点1: (50, 0) —— 把曲线从起点向上拉。
- 控制点2: (150, 200) —— 把曲线向终点向下拉。
- 终点: (200, 100)

### 平滑连接 (`S`, `s`)

当你需要连接多段贝塞尔曲线，并希望连接处平滑过渡时，可以使用 `S` 命令。

- **格式**: `S x2 y2, x y`
- **原理**: `S` 命令会自动计算第一个控制点，使其成为上一段曲线第二个控制点的**中心对称点**。这保证了曲线在连接处的斜率连续。

## 4.3.2 二次贝塞尔曲线 (`Q`, `q`, `T`, `t`)

二次贝塞尔曲线比三次简单，它只需要**一个控制点**和一个**终点**。

- **格式**: `Q x1 y1, x y`
- **参数**:
    - `(x1, y1)`: 唯一的控制点。
    - `(x, y)`: 终点。

### 交互演示：Q 命令

二次贝塞尔曲线通常用于绘制简单的弧形。

<div className="my-8">
  <PathDemo initialD="M 10 100 Q 100 0 200 100" />
</div>

- 起点: (10, 100)
- 控制点: (100, 0) —— 把曲线向上拉。
- 终点: (200, 100)

### 平滑连接 (`T`, `t`)

类似于 `S`，`T` 命令用于平滑连接二次贝塞尔曲线。

- **格式**: `T x y`
- **原理**: 自动推断控制点为上一段控制点的中心对称点。

```xml
<!-- 连续的波浪 -->
<path d="M 10 100 Q 50 0 100 100 T 190 100" stroke="black" fill="none"/>
```

## 4.3.3 总结与对比

| 类型 | 命令 | 控制点数量 | 灵活性 | 计算复杂度 |
| :--- | :--- | :--- | :--- | :--- |
| **二次 (Quadratic)** | `Q`, `T` | 1 | 较低，适合简单弧线 | 低 |
| **三次 (Cubic)** | `C`, `S` | 2 | 极高，可绘制复杂 S 形 | 高 |

在实际设计软件（如 Illustrator, Sketch）中，钢笔工具绘制的通常是**三次贝塞尔曲线**。
