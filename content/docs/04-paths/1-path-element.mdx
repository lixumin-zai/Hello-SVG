# 4.1 `<path>` 元素与 `d` 属性

`<path>` 元素是 SVG 中最强大的工具。它可以用来创建线条、曲线、弧形等等。从本质上讲，`<path>` 是所有其他图形（如 `<rect>`, `<circle>`, `<polygon>` 等）的超集。

## 4.1.1 什么是 `d` 属性？

`<path>` 元素的形状是通过属性 `d` 来定义的。这里的 "d" 代表 **Data**（数据）。

`d` 属性包含了一系列命令和参数，这些命令告诉浏览器“画笔”应该如何移动。你可以把 `d` 属性想象成一段控制绘图机器人的程序代码。

### 命令格式

- **命令字母**：每个命令由一个字母表示（如 `M` 代表 Move to）。
- **参数**：命令后面跟着一组坐标参数。
- **大小写敏感**：
    - **大写字母**（如 `M`）：表示 **绝对坐标**（Absolute），基于 SVG 画布的原点 (0,0)。
    - **小写字母**（如 `m`）：表示 **相对坐标**（Relative），基于上一个点的坐标。

## 4.1.2 第一个路径

让我们来看一个最简单的路径示例：

```xml
<path d="M 10 10 L 50 50" stroke="black" />
```

这段代码的意思是：
1.  `M 10 10`: 将画笔移动到绝对坐标 (10, 10)。
2.  `L 50 50`: 从当前位置画一条直线到绝对坐标 (50, 50)。

## 4.1.3 命令概览

在接下来的章节中，我们将详细学习这些命令：

1.  **移动与直线**
    *   `M` / `m`: Move to (移动)
    *   `L` / `l`: Line to (画线)
    *   `H` / `h`: Horizontal Line to (水平线)
    *   `V` / `v`: Vertical Line to (垂直线)
    *   `Z` / `z`: Close Path (闭合路径)
2.  **贝塞尔曲线**
    *   `C` / `c`: Cubic Bezier (三次贝塞尔曲线)
    *   `S` / `s`: Smooth Cubic Bezier (平滑三次贝塞尔曲线)
    *   `Q` / `q`: Quadratic Bezier (二次贝塞尔曲线)
    *   `T` / `t`: Smooth Quadratic Bezier (平滑二次贝塞尔曲线)
3.  **弧线**
    *   `A` / `a`: Elliptical Arc (椭圆弧)
