# Part 4: 路径 (Path) —— SVG 的核心

如果说 SVG 是矢量图形的皇冠，那么 `<path>` 元素就是皇冠上的明珠。它是 SVG 中最强大、最灵活的元素。

虽然我们已经学习了 `<rect>`、`<circle>` 等基础形状，但它们本质上都是 `<path>` 的语法糖。`<path>` 可以绘制任意形状：从简单的直线、圆弧，到复杂的贝塞尔曲线，甚至是手绘的笔迹。

在本章中，我们将系统地解构 `<path>` 元素的 `d` 属性（Data），掌握它所包含的各种绘图命令：

1.  **移动与直线**：`M`, `L`, `H`, `V`, `Z`
2.  **贝塞尔曲线**：`C`, `S`, `Q`, `T`
3.  **弧线**：`A`

掌握了路径，你就掌握了 SVG 的精髓。
