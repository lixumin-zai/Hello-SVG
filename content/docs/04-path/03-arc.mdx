## A 命令 (椭圆弧)

**语法：`A rx ry x-axis-rotation large-arc-flag sweep-flag x y`**

它是所有 SVG 路径命令中参数最多的（足足有 7 个！）。为什么画个弧线这么复杂？
因为在平面几何中，如果**已知起点和终点，并给定一个椭圆的大小，可以画出 4 条完全不同的弧线连接这两点**。

我们需要用这 7 个参数来唯一确定我们想要的那条线。

### 参数速记指南
- `rx`, `ry`：椭圆的横向和纵向半径。如果是画正圆的弧线，这两个值相等即可。
- `x-axis-rotation`：整个椭圆绕 X 轴旋转的角度。
- **`large-arc-flag` (大弧标志)**：
  - `0` = 选短的那条路（小弧）
  - `1` = 选长的那条路（大弧）
- **`sweep-flag` (扫扫标志 / 方向标志)**：
  - `0` = 逆时针画
  - `1` = 顺时针画
- `x`, `y`：弧线的终点坐标。

<Callout type="info" title="画一个完整的圆？">
  **A 命令无法画出一个完整的圆！** 因为如果起点和终点重合，数学上有无数个解。如果要用 `path` 画圆，必须用两个半圆弧（A 命令）拼接起来，或者干脆用 `<circle>` 元素。
</Callout>

这由 `fill-rule` 属性决定。
