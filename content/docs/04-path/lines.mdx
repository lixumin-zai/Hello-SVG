最简单的路径就是一系列的直线。它就像我们在纸上画图一样，先把笔移动到某个点，然后画线到另一个点。

## 4.2.1 路径构建器：点与线

这是一个交互式的路径构建器。点击画布来添加点，观察下方的路径数据（d 属性）是如何生成的。

<PathBuilder />

### 关键命令详解

1.  **`M` (MoveTo)**：`M x y`
    -   拿起笔，移动到 `(x, y)`，但不画线。通常用于路径的**起点**。
    -   如果 `M` 后面跟着多个点，后续的点会被视为 `L`（LineTo）。

2.  **`L` (LineTo)**：`L x y`
    -   从当前点画一条直线到 `(x, y)`。

3.  **`H` (Horizontal LineTo)**：`H x`
    -   画一条水平线到 `x` 坐标。`y` 坐标保持不变。
    -   相当于 `L x currentY`。
    -   非常适合绘制矩形或网格。

4.  **`V` (Vertical LineTo)**：`V y`
    -   画一条垂直线到 `y` 坐标。`x` 坐标保持不变。

5.  **`Z` (ClosePath)**：`Z`
    -   闭合路径。从当前点画一条直线回到路径的**起始点**（即最近的一个 `M` 命令的位置）。
    -   如果你只是用 `L` 画回起点，那个角可能看起来不平滑（没有正确连接）。使用 `Z` 可以让首尾完美闭合。

## 练习：手写一个三角形

试着写出这样一个三角形的路径：
1.  起点在 (50, 10)。
2.  画线到 (90, 90)。
3.  画线到 (10, 90)。
4.  闭合。
