SVG 中最复杂但也最强大的命令莫过于 **`A`**（Arc）。它允许你绘制椭圆的一段弧线。

## 4.4.1 A 命令的语法

```xml
A rx ry x-axis-rotation large-arc-flag sweep-flag x y
```

这 7 个参数让无数初学者望而却步。但只要拆解开来，其实很简单：

1.  **`rx`, `ry`**：椭圆的 X 轴和 Y 轴半径。决定了椭圆的大小。
2.  **`x-axis-rotation`**：椭圆相对于 X 轴的旋转角度。
3.  **`large-arc-flag`**：决定取**大弧**（1）还是**小弧**（0）。
4.  **`sweep-flag`**：决定**顺时针**（1）还是**逆时针**（0）画弧。
5.  **`x`, `y`**：弧线的**终点坐标**。

### 关键概念：大弧 vs. 小弧

给定两个点（起点和终点）和一个椭圆半径，实际上有 **4 种**可能的弧线可以连接这两个点：

1.  **大弧 + 顺时针**
2.  **大弧 + 逆时针**
3.  **小弧 + 顺时针**
4.  **小弧 + 逆时针**

这就是为什么我们需要 `large-arc-flag` 和 `sweep-flag` 来明确告诉浏览器我们要画哪一段。

### 实战：绘制饼图扇形

利用 `A` 命令，我们可以非常容易地绘制饼图的一个扇形。

```xml
<path d="M 50 50 L 50 10 A 40 40 0 0 1 90 50 Z" fill="orange" />
```

-   **起点**：(50, 50) 圆心。
-   **线段**：画到 (50, 10) —— 圆的正上方。
-   **弧线**：画半径为 40 的圆弧，到 (90, 50) —— 圆的正右方。顺时针（1），小弧（0）。
-   **闭合**：回到圆心。

## 总结

-   **M, L, Z**：画直线。
-   **Q, C**：画贝塞尔曲线（平滑）。
-   **A**：画圆弧（椭圆的一部分）。
-   **T, S**：画平滑连接的曲线。
-   **H, V**：水平/垂直线的简写。

掌握了这些，你就掌握了 SVG 绘图的核心。
