# 3.3 圆与椭圆

圆和椭圆是 SVG 中常见的曲线图形。SVG 提供了 `<circle>` 和 `<ellipse>` 元素来分别绘制圆形和椭圆形。

## 3.3.1 圆 (`<circle>`)

`<circle>` 元素用于绘制正圆。它通过圆心坐标和半径来定义。

- `cx`: 圆心的 x 坐标。
- `cy`: 圆心的 y 坐标。
- `r`: 圆的半径（必须大于等于 0）。

### 交互演示：Circle

调整圆心位置和半径，观察圆形的变化。

<div className="my-8">
  <ShapePlayground initialType="circle" />
</div>

### 代码示例

```xml
<svg width="200" height="200" xmlns="http://www.w3.org/2000/svg">
  <!-- 一个红色的圆形 -->
  <circle cx="100" cy="100" r="80" fill="red" />
</svg>
```

## 3.3.2 椭圆 (`<ellipse>`)

`<ellipse>` 元素用于绘制椭圆。它通过中心坐标和两个半径（x 方向和 y 方向）来定义。

- `cx`: 椭圆中心的 x 坐标。
- `cy`: 椭圆中心的 y 坐标。
- `rx`: 椭圆 x 方向的半径。
- `ry`: 椭圆 y 方向的半径。

### 交互演示：Ellipse

调整 `rx` 和 `ry`，观察椭圆的形状变化。当 `rx` 和 `ry` 相等时，椭圆就是一个圆。

<div className="my-8">
  <ShapePlayground initialType="ellipse" />
</div>

### 代码示例

```xml
<svg width="200" height="200" xmlns="http://www.w3.org/2000/svg">
  <!-- 一个绿色的椭圆 -->
  <ellipse cx="100" cy="100" rx="80" ry="50" fill="green" />
</svg>
```

### 注意事项

1.  **圆心坐标**：如果不指定 `cx` 和 `cy`，默认值为 0，这意味着圆心位于 SVG 画布的左上角，只能看到 1/4 的圆形。
2.  **半径限制**：如果 `r`、`rx` 或 `ry` 为 0，图形将不可见。如果为负数，通常是错误的。
3.  **圆是椭圆的特例**：`<circle cx="50" cy="50" r="30" />` 等价于 `<ellipse cx="50" cy="50" rx="30" ry="30" />`。但在语义上，使用 `<circle>` 更清晰。
