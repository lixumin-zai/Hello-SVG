# 3.4 多边形

SVG 中的 `<polygon>` 元素用于绘制封闭的多边形，它和 `<polyline>` 非常相似，但有一个关键区别：`<polygon>` 会自动将最后一个点连接到第一个点，形成封闭的形状。

## 3.4.1 多边形 (`<polygon>`)

`<polygon>` 元素用于定义由多条直线段组成的封闭形状。它主要通过 `points` 属性来定义一系列的顶点。

- `points`: 一系列 x,y 坐标对，用空格或逗号分隔。例如："50,150 100,50 150,150"。
- `fill`: 填充颜色。通常设置为 `none`，如果不设置，多边形内部将被填充。
- `fill-rule`: 控制多边形内部填充规则（如 `nonzero` 或 `evenodd`）。

### 代码示例

```xml
<svg width="200" height="200" xmlns="http://www.w3.org/2000/svg">
  <!-- 一个简单的三角形 -->
  <polygon points="50,150 100,50 150,150" fill="lime" stroke="purple" stroke-width="2" />
  
  <!-- 一个五角星 -->
  <polygon points="100,10 40,198 190,78 10,78 160,198" fill="gold" stroke="purple" stroke-width="5" fill-rule="evenodd" />
</svg>
```

### 填充规则 (`fill-rule`)

当多边形自身相交时，`fill-rule` 属性决定了哪些区域属于多边形的“内部”。
- `nonzero` (默认): 基于射线与路径段相交的方向计算 winding number。
- `evenodd`: 基于射线与路径段相交的次数计算奇偶性。

### 注意事项

1.  **闭合性**：`<polygon>` 总是自动闭合的，即使你在 `points` 中没有显式回到起点。
2.  **点顺序**：点的顺序决定了绘制的方向，对于复杂的自相交多边形，顺序会影响填充结果（结合 `fill-rule`）。
3.  **坐标对格式**：推荐使用 `x,y x,y` 的格式，可读性更好。
