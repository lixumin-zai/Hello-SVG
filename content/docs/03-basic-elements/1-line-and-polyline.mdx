# 3.1 直线与折线

直线是所有几何图形中最基本的元素。SVG 提供了 `<line>` 元素来绘制简单的直线段，以及 `<polyline>` 元素来绘制由多条直线连接而成的折线。

## 3.1.1 直线 (`<line>`)

`<line>` 元素用于绘制连接两个点的直线段。它主要通过以下属性来定义：

- `x1`: 起点的 x 坐标。
- `y1`: 起点的 y 坐标。
- `x2`: 终点的 x 坐标。
- `y2`: 终点的 y 坐标。
- `stroke`: 线条的颜色（必须指定，否则不可见）。
- `stroke-width`: 线条的宽度（可选，默认为 1）。

### 交互演示：Line

试着拖动滑块，改变直线的起点和终点，观察坐标的变化。

<div className="my-8">
  <ShapePlayground initialType="line" />

### 代码示例

```xml
<svg width="200" height="200" xmlns="http://www.w3.org/2000/svg">
  <!-- 一条简单的红线 -->
  <line x1="50" y1="50" x2="150" y2="150" stroke="red" stroke-width="2" />
</svg>
```

## 3.1.2 折线 (`<polyline>`)

当需要绘制连续的多条线段时，`<polyline>` 是更好的选择。它只用一个 `points` 属性就可以定义一系列的点，SVG 会按顺序连接这些点。

- `points`: 一系列 x,y 坐标对，用空格或逗号分隔。例如："50,50 100,100 150,50"。
- `fill`: 填充颜色。对于开放的折线，通常设置为 `none`，否则 SVG 会尝试填充起始点和终点构成的区域。
- `stroke`: 线条颜色。

### 交互演示：Polyline

修改下面的 `points` 属性，看看折线是如何变化的。

<div className="my-8">
  <ShapePlayground initialType="polyline" />
</div>

### 代码示例

```xml
<svg width="300" height="200" xmlns="http://www.w3.org/2000/svg">
  <!-- 一个未填充的折线图 -->
  <polyline 
    points="20,100 40,60 70,80 100,20 130,120" 
    fill="none" 
    stroke="blue" 
    stroke-width="2" 
  />
</svg>
```

### 注意事项

1.  **坐标对格式**：虽然可以使用逗号分隔 x 和 y，也可以全部使用空格，但推荐 `x,y x,y` 的格式，可读性更好。
2.  **填充行为**：`<polyline>` 默认会有黑色填充。如果你只想画线，记得设置 `fill="none"`。
3.  **连接处样式**：可以使用 `stroke-linejoin` 属性控制线段连接处的样式（如尖角 `miter`、圆角 `round`、斜角 `bevel`），这将在"描边"章节详细介绍。
