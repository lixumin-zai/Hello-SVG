SVG 提供了一组基本的形状元素，通过这些元素，我们可以构建出绝大多数常见的图形。虽然 `<path>` 元素可以绘制任意形状，但使用这些预定义的形状元素通常更直观、语义更清晰，也更容易通过 CSS 进行样式控制。

在本章中，我们将深入学习以下基础图形元素：

1.  **直线与折线 (`<line>`, `<polyline>`)**：绘制简单的线段和连续的折线。
2.  **矩形 (`<rect>`)**：绘制矩形及圆角矩形。
3.  **圆与椭圆 (`<circle>`, `<ellipse>`)**：绘制圆形和椭圆形。
4.  **多边形 (`<polygon>`)**：绘制封闭的多边形。
5.  **文本 (`<text>`)**：在 SVG 中渲染文本内容。
