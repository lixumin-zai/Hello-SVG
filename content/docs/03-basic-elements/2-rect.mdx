# 3.2 矩形

矩形是使用最广泛的 SVG 形状之一，不仅可以用来绘制矩形，还可以用来绘制正方形和圆角矩形。SVG 使用 `<rect>` 元素来创建矩形。

## 3.2.1 矩形 (`<rect>`)

`<rect>` 元素用于绘制矩形。它定义了矩形的左上角坐标、宽度和高度。

- `x`: 矩形左上角的 x 坐标。
- `y`: 矩形左上角的 y 坐标。
- `width`: 矩形的宽度（必须大于等于 0）。
- `height`: 矩形的高度（必须大于等于 0）。
- `rx`: 矩形角的 x 方向半径（可选，用于圆角）。
- `ry`: 矩形角的 y 方向半径（可选，用于圆角）。

### 代码示例

```xml
<svg width="200" height="200" xmlns="http://www.w3.org/2000/svg">
  <!-- 一个简单的红色矩形 -->
  <rect x="10" y="10" width="100" height="80" fill="red" />
  
  <!-- 一个蓝色边框的圆角矩形 -->
  <rect x="120" y="10" width="60" height="80" rx="10" ry="10" fill="none" stroke="blue" />
</svg>
```

### 注意事项

1.  **圆角半径**：
    *   如果只设置 `rx`，`ry` 默认等于 `rx`。
    *   如果 `rx` 大于宽度的一半，它会被限制为宽度的一半。
    *   `rx` 和 `ry` 共同决定圆角的形状（椭圆弧）。
2.  **宽高限制**：
    *   如果 `width` 或 `height` 为 0，矩形将不可见。
    *   如果 `width` 或 `height` 为负数，是一个错误，通常会被忽略或报错。
3.  **默认值**：如果不指定 `x` 和 `y`，默认值为 0。
