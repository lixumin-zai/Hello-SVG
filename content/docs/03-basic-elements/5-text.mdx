# 3.5 文本

SVG 中的 `<text>` 元素用于定义和显示文本内容。SVG 文本不仅可以进行复杂的样式控制，还可以作为图形元素进行变换、填充、描边等操作。

## 3.5.1 文本 (`<text>`)

`<text>` 元素用于在 SVG 中显示文本。它定义了文本的起始位置和内容。

- `x`: 文本起始位置的 x 坐标。
- `y`: 文本起始位置的 y 坐标。
- `font-family`: 字体系列。
- `font-size`: 字体大小。
- `font-weight`: 字体粗细。
- `text-anchor`: 文本锚点（`start`、`middle`、`end`），控制文本对齐方式。
- `fill`: 文本填充颜色（默认为黑色）。
- `stroke`: 文本描边颜色。

### 代码示例

```xml
<svg width="200" height="100" xmlns="http://www.w3.org/2000/svg">
  <!-- 一个简单的文本 -->
  <text x="10" y="50" font-family="Verdana" font-size="35" fill="blue">Hello, SVG!</text>
  
  <!-- 描边文本 -->
  <text x="10" y="90" font-family="Verdana" font-size="35" fill="none" stroke="black">Stroked text</text>
</svg>
```

### 文本属性

SVG 文本可以使用大部分 CSS 文本属性，如：
- `font-family`
- `font-size`
- `font-weight`
- `font-style`
- `text-decoration`
- `letter-spacing`
- `word-spacing`

此外，还有 SVG 特有的属性：
- `text-anchor`: 控制文本水平对齐（`start` 左对齐, `middle` 居中, `end` 右对齐）。
- `dominant-baseline`: 控制文本垂直对齐（`auto`, `middle`, `hanging` 等）。

### 路径文本 (`<textPath>`)

SVG 还支持将文本沿着一条路径排列，这通过 `<textPath>` 元素实现（将在后续章节详细介绍）。

### 注意事项

1.  **文本选择**：SVG 文本默认是可选中的，这一点优于 Canvas 绘制的文本。
2.  **字体加载**：SVG 依赖系统字体或通过 `@font-face` 引入的 Web 字体。如果字体不可用，将使用回退字体。
3.  **无障碍性**：SVG 文本对于屏幕阅读器是可见的，这使得 SVG 图表具有很好的无障碍性。
