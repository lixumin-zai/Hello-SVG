## 现代前端框架中的 SVG 组件化 (SVGR)

在 React 或 Vue 中，我们其实很少去手写 `<svg><use .../></svg>` 这样的底层代码了。
现代工具链（如 **SVGR**）可以在编译时，自动将你引入的 `.svg` 文件转换成一个 React 组件。

```jsx
// Webpack / Vite 配置好 SVGR 插件后：
import { ReactComponent as HeartIcon } from './heart.svg';

export function App() {
  return (
    <button>
      <HeartIcon className="w-6 h-6 text-red-500" />
      点赞
    </button>
  );
}
```
这种方式彻底屏蔽了 SVG 的底层细节，让图标变成了普通的 UI 组件，同时天然享受 Tree-shaking 带来的体积优化。
