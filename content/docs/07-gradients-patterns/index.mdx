渐变是让扁平的 2D 图形产生 3D 光影错觉最有效的方法。SVG 提供了两种基础渐变：**线性渐变 (`<linearGradient>`)** 和 **径向渐变 (`<radialGradient>`)**。

与 CSS 的 `background: linear-gradient(...)` 不同，SVG 的渐变必须先在 `<defs>` 标签中“定义”，然后通过 `id` 在图形的 `fill` 或 `stroke` 属性中“引用”。

<Callout type="tip" title="渐变的坐标系 (gradientUnits)">
  默认情况下，渐变的坐标（如 100%）是相对于**应用它的那个图形（Bounding Box）**来计算的。这意味着同一个渐变应用在不同大小的矩形上，会自动拉伸适应。
  <br/>如果你想让渐变相对于整个 SVG 画布来计算，可以设置 <code>gradientUnits="userSpaceOnUse"</code>。
</Callout>
