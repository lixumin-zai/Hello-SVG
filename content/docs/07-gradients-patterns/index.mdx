渐变是让扁平的 2D 图形产生 3D 光影错觉最有效的方法。SVG 提供了两种基础渐变：**线性渐变 (`<linearGradient>`)** 和 **径向渐变 (`<radialGradient>`)**。

与 CSS 的 `background: linear-gradient(...)` 不同，SVG 的渐变必须先在 `<defs>` 标签中“定义”，然后通过 `id` 在图形的 `fill` 或 `stroke` 属性中“引用”。

## 核心概念

### 1. 线性渐变的方向向量

在线性渐变中，我们通过 `(x1, y1)` 和 `(x2, y2)` 来定义一条直线（方向向量）。渐变颜色将沿着这条线均匀过渡。

<Cards>
  <Card title="水平渐变（默认）" description='x1="0%", y1="0%" x2="100%", y2="0%"' />
  <Card title="垂直渐变" description='x2="0%", y2="100%"' />
  <Card title="对角线渐变" description='x2="100%", y2="100%"' />
</Cards>

### 2. 径向渐变的神奇焦点 (fx, fy)

径向渐变是从中心向外呈圆形（或椭圆形）扩散的渐变。

*   `cx, cy, r`：定义了渐变的最外层边界（即 100% 颜色所在的位置）。
*   **`fx, fy` (Focus 焦点)**：这是径向渐变最强大的属性。它定义了 0% 颜色开始的位置。如果焦点偏离了中心点，你就可以轻松画出一个**带有立体高光的球体**（见上方演示器的径向渐变选项卡，尝试拖动 `fx, fy` 滑块）。

<Callout type="tip" title="渐变的坐标系 (gradientUnits)">
  默认情况下，渐变的坐标（如 100%）是相对于**应用它的那个图形（Bounding Box）**来计算的。这意味着同一个渐变应用在不同大小的矩形上，会自动拉伸适应。
  <br/>如果你想让渐变相对于整个 SVG 画布来计算，可以设置 <code>gradientUnits="userSpaceOnUse"</code>。
</Callout>
