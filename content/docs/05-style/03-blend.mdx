除了简单的透明度（Opacity），SVG 还完全支持 CSS 的合成与混合规范。这使得我们可以直接在浏览器中实现复杂的光影叠加、正片叠底等“设计师级”的视觉效果，而不需要导出庞大的 PNG 图片。

<Callout type="tip" title="何时使用 isolation？">
  当你设计了一个由多个互相混合的形状组成的复杂 Logo 或组件，你不希望它放到用户的网页上时，因为受到用户网页深色/浅色背景的影响而改变原本的颜色，这时就必须给最外层的 <code>&lt;g&gt;</code> 加上 <code>isolation: isolate</code>。
</Callout>
