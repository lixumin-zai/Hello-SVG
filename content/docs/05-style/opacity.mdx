在 SVG 中，透明度（Transparency）是一个复杂的话题。

## 5.4.1 属性透明度 vs. 全局透明度

SVG 有两种设置透明度的方法：

1.  **全局透明度 (`opacity`)**：应用于整个元素及其所有子元素。
2.  **属性透明度 (`fill-opacity`, `stroke-opacity`)**：只影响特定的绘制操作。

## 5.4.2 为什么我的圆看起来有点脏？

如果你在绘制重叠图形时遇到了奇怪的颜色，通常是因为你**错误地使用了 opacity**。

-   **错误用法**：`opacity="0.5"`。这会让填充和描边都变半透明，甚至还会让它们重叠的部分变深（因为它们是独立的图层）。
-   **正确用法**：使用 `fill-opacity` 和 `stroke-opacity`。这样填充和描边是分别计算的，不会发生重叠变深的问题。

## 5.4.3 混合模式 (Blend Modes)

SVG 也是支持 CSS 混合模式的！

```css
.my-shape {
  mix-blend-mode: multiply; /* 正片叠底 */
}
```

这使得你可以用 SVG 实现类似 Photoshop 的效果，比如颜色加深、滤色、叠加等。

### 常用混合模式

-   **`normal`**：默认。遮挡下方图层。
-   **`multiply`**：变暗。像马克笔涂色。
-   **`screen`**：变亮。像投影仪叠加。
-   **`overlay`**：增加对比度。亮处更亮，暗处更暗。
