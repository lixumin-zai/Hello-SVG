SVG 变换（Transforms）允许你改变对象的位置、大小和方向。它是制作 SVG 动画和复用组件的核心。

## 本章概览

1.  **坐标变换基础**：理解“用户坐标系”是如何被扭曲的。
2.  **常用命令**：`translate`, `rotate`, `scale`, `skew`。
3.  **矩阵操作**：`matrix` —— 一切变换的终极形式。
4.  **嵌套与分组**：利用 `<g>` 实现复杂的机械臂运动。

## 6.1 变换的本质

在 SVG 中，当你对一个物体应用变换时，你实际上是在**改变它所处的坐标系**。

-   **平移**：移动原点 (0,0) 的位置。
-   **旋转**：旋转坐标轴。
-   **缩放**：拉伸坐标轴的刻度。

## 6.2 变换属性语法

SVG 的 `transform` 属性可以包含一个或多个命令，用空格或逗号分隔。

```xml
<rect transform="translate(50, 50) rotate(45) scale(2)" />
```

### 变换顺序的重要性

**顺序非常重要！**
SVG 变换是按照**从左到右**的顺序应用于坐标系的。

1.  `translate(50, 50) rotate(45)`：先移动原点到 (50,50)，然后绕新原点旋转 45 度。
2.  `rotate(45) translate(50, 50)`：先绕 (0,0) 旋转坐标轴，然后沿着**歪斜的轴**移动 (50,50)。结果完全不同！

通常我们推荐的顺序是：**先平移（Translate），后旋转（Rotate），最后缩放（Scale）**。
