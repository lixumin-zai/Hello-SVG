# 6.1 SVG transform 属性

`transform` 属性允许我们修改元素及其子元素的坐标系。它包含了一系列变换函数，可以组合使用。

## 6.1.1 平移 (`translate`)

`translate(tx, ty)` 将坐标系原点移动到 `(tx, ty)`。

- 如果省略 `ty`，默认为 0。
- 例如：`translate(50, 50)` 将所有内容的坐标都加 50。

## 6.1.2 旋转 (`rotate`)

`rotate(angle, [cx, cy])` 将坐标系旋转 `angle` 度。

- **默认原点**：旋转默认围绕当前用户坐标系的原点 `(0, 0)` 进行。这与 CSS transform 不同（CSS 默认围绕元素中心）。
- **指定中心**：你可以提供可选的 `cx, cy` 参数，指定旋转中心。例如 `rotate(45, 50, 50)` 围绕点 `(50, 50)` 旋转 45 度。这实际上是 `translate(50, 50) rotate(45) translate(-50, -50)` 的简写。

## 6.1.3 缩放 (`scale`)

`scale(sx, [sy])` 将坐标系按比例缩放。

- 如果省略 `sy`，默认等于 `sx`（等比缩放）。
- **注意**：缩放不仅会改变图形大小，也会改变位置（坐标值也会被缩放）和描边宽度。

## 6.1.4 倾斜 (`skewX`, `skewY`)

- `skewX(angle)`: 沿 X 轴倾斜，保持 Y 坐标不变。
- `skewY(angle)`: 沿 Y 轴倾斜，保持 X 坐标不变。
