# Part 6: 变换 (Transforms)

在 SVG 中，变换（Transform）不仅仅是移动或旋转图形，它是改变**坐标系**本身的一种方式。

当你对一个元素应用变换时，你实际上是在告诉浏览器：“请为这个元素建立一个新的坐标系，这个新坐标系是基于原坐标系经过平移/旋转/缩放得到的。”

在本章中，我们将深入研究：

1.  **SVG transform 属性**：基本的平移、旋转、缩放和倾斜。
2.  **变换的数学基础**：理解矩阵乘法如何影响变换顺序。
3.  **CSS vs SVG**：理解两种变换系统的关键差异（尤其是原点问题）。
4.  **嵌套变换**：如何利用分组 (`<g>`) 管理复杂的坐标系。

掌握变换是制作 SVG 动画和复杂图形交互的基石。
