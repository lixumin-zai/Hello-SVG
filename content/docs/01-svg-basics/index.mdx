## 什么是 SVG？

SVG 的全称是 **Scalable Vector Graphics**（可缩放矢量图形），它是一种基于 XML 的标记语言，用于描述二维的矢量图形。

<SvgDefinition />

## SVG 的诞生与标准化历程

在 SVG 出现之前，Web 矢量图形标准处于分裂状态（微软主推 VML，Adobe 主推 PGML）。为了统一标准，W3C 成立了专门的工作组，吸收了两者的优点，起草了全新的 SVG 标准。

## SVG vs. Flash：一场时代之战

在 SVG 缓慢普及的年代，Adobe 的 **Flash** 曾统治了 Web 交互与矢量动画领域。但随着移动互联网时代的到来，开放、标准化的 SVG 结合 HTML5，最终成为了这场时代之战的赢家。
