# Part 5: 样式与外观

SVG 不仅仅是几何形状的集合，它还拥有丰富的样式系统。你可以像控制 HTML 元素一样控制 SVG 的颜色、边框、透明度等外观。

在本章中，我们将深入探索 SVG 的样式世界：

1.  **样式的三种方式**：属性、内联样式、外部 CSS。
2.  **填充 (Fill)**：掌握颜色填充的细节。
3.  **描边 (Stroke)**：学习如何控制线条的粗细、端点样式、连接方式以及虚线效果。
4.  **透明度 (Opacity)**：理解整体透明度与填充/描边透明度的区别。
5.  **CSS 特性**：探索 SVG 独有的 CSS 属性。
