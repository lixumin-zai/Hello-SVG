<Callout type="warn" title="use 的限制">
  <code>&lt;use&gt;</code> 生成的是 Shadow DOM，你无法用 CSS 或 JS 深入进去修改汽车轮胎的颜色。
  它最大的痛点是：**你无法改变原图形的宽高比例！** 哪怕你在 <code>&lt;use&gt;</code> 上写了 <code>width="100"</code>，原图形也只会被生硬地裁剪，而不是等比缩放。
</Callout>
