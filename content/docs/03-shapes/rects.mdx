矩形是 UI 设计中最基础的形状，SVG 提供了 `<rect>` 元素来绘制它。

## 3.2.1 `<rect>`：长宽与位置

```xml
<rect x="10" y="10" width="80" height="60" />
```

- **必须属性**：
  - `x`, `y`：左上角坐标。如果不写，默认为 0。
  - `width`, `height`：宽高。**如果设为 0，矩形不可见**。
- **填充**：默认填充黑色，描边无。

## 3.2.2 圆角矩形：rx 与 ry

这是最让人头疼的部分：**圆角**。

```xml
<rect rx="10" ry="10" width="100" height="100" />
```

- **`rx`**：圆角在 X 轴的半径。
- **`ry`**：圆角在 Y 轴的半径。

### 圆角规则：
1.  **rx = ry**：标准的圆角（如按钮）。你只需要写 `rx`，`ry` 默认为 `rx`。
2.  **rx ≠ ry**：椭圆角（较少见，除非为了艺术效果）。
3.  **rx > width/2**：如果半径超过一半，它会变成半圆（胶囊形）。

### 为什么不用 `border-radius`？

CSS 的 `border-radius` 是作用在 HTML 元素盒子上的。SVG 元素内部的形状是由几何属性决定的。
虽然 SVG 2.0 提议允许 CSS 设置 `rx` 和 `ry`，但目前的浏览器支持度不一。
**最佳实践**：始终使用 `rx` 和 `ry` 属性来定义圆角。

## 实战：用矩形绘制图表

矩形是柱状图的基础。

```xml
<svg width="200" height="100">
  <rect x="10" y="10" width="30" height="80" fill="blue" />
  <rect x="50" y="30" width="30" height="60" fill="red" />
  <rect x="90" y="50" width="30" height="40" fill="green" />
</svg>
```

- **Y 轴倒置**：注意 SVG 的 Y 轴向下。如果你想让柱子从底部向上生长，你需要手动计算 `y` 值（`y = 容器高度 - 柱子高度`）。这是新手最容易犯错的地方。
