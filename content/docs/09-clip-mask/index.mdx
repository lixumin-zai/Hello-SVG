有时，我们需要隐藏图形的一部分，或者只显示特定区域。SVG 提供了两种机制来实现这一点：**裁剪路径（Clip Path）**和**遮罩（Mask）**。

虽然它们的目的相似，但工作原理完全不同。

## 本章概览

1.  **Clip Path**：用矢量形状去“切”图形。硬边缘，非黑即白。
2.  **Mask**：用像素亮度去“遮”图形。软边缘，支持半透明。
3.  **对比实验**：直观感受两者的区别。

## 9.2 核心区别速查

| 特性 | Clip Path (裁剪) | Mask (遮罩) |
| :--- | :--- | :--- |
| **原理** | 几何切割 | 亮度映射 |
| **边缘** | 硬边缘 (矢量) | 软边缘 (像素/渐变) |
| **半透明** | 不支持 | 支持 (灰度) |
| **性能** | 极高 (浏览器只需计算轮廓) | 较低 (涉及像素混合) |
| **适用场景** | 头像裁剪、形状限制 | 渐变消失、纹理叠加 |
