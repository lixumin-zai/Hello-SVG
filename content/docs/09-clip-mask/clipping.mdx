## 9.1.1 定义与引用

`<clipPath>` 元素定义了一个或多个形状。这些形状的**并集**构成了可见区域。形状之外的所有内容都会被隐藏。

```xml
<defs>
  <clipPath id="myClip">
    <circle cx="100" cy="100" r="50" />
    <rect x="0" y="0" width="100" height="100" />
  </clipPath>
</defs>

<!-- 使用裁剪 -->
<image href="photo.jpg" width="200" height="200" clip-path="url(#myClip)" />
```

## 9.1.2 裁剪规则

-   **非黑即白**：一个像素要么在形状内（显示），要么在形状外（隐藏）。不存在半透明。
-   **无视颜色**：`<clipPath>` 内的形状颜色（fill, stroke）会被完全忽略。只有几何轮廓起作用。
-   **叠加**：你可以放多个形状，它们会叠加在一起。

## 9.1.3 CSS `clip-path`

你也可以在 CSS 中直接使用 `clip-path`，甚至不需要 SVG 文件。

```css
.avatar {
  clip-path: circle(50% at 50% 50%);
}
```

或者引用 SVG 中的定义：

```css
.avatar {
  clip-path: url(#myClip);
}
```

这在做响应式 UI（如多边形卡片）时非常有用。
