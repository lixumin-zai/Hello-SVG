## 9.2.1 什么是 Mask？

Mask 是基于**亮度（Luminance）**的透明度映射。

-   **白色**：完全不透明（显示）。
-   **黑色**：完全透明（隐藏）。
-   **灰色**：半透明。

这意味着，你可以使用**渐变**、**图片**、甚至**另一个 SVG 元素**来作为遮罩。

```xml
<defs>
  <mask id="fade">
    <!-- 从白到黑的线性渐变 -->
    <rect width="100%" height="100%" fill="url(#gradient)" />
  </mask>
</defs>

<!-- 使用遮罩 -->
<rect width="200" height="200" fill="red" mask="url(#fade)" />
```

## 9.2.2 遮罩的高级用法

### 1. 纹理遮罩

你可以用一张黑白纹理图片（如噪点图）作为遮罩，给任何元素添加磨损效果。

### 2. 文字遮罩

用文字作为遮罩，可以让背景图片透过文字显示出来（Text Clipping Effect）。

```xml
<mask id="textMask">
  <rect width="100%" height="100%" fill="black" />
  <text x="50" y="50" fill="white" font-size="50">SVG</text>
</mask>

<image href="scenery.jpg" mask="url(#textMask)" />
```

这里背景必须填黑（完全隐藏），文字填白（完全显示），才能实现镂空效果。

### 3. `mask-type`

SVG 2.0 引入了 `mask-type` 属性：
-   `luminance`（默认）：基于亮度。
-   `alpha`：基于透明度（这就跟 CSS mask 类似了）。

但为了兼容性，目前主要还是使用亮度模式。
