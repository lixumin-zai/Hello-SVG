## 8.3.1 `<feGaussianBlur>`：高斯模糊

最基础的滤镜。

-   **`stdDeviation`**：标准差。控制模糊的程度。可以是一个值（如 `5`），也可以是两个值（如 `5 10`，分别控制 X 轴和 Y 轴的模糊）。

## 8.3.2 `<feDropShadow>`：投影

SVG 1.1 中做投影需要复杂的 `<feOffset>` + `<feGaussianBlur>` + `<feBlend>` 组合。但现在我们有了现代的简写版：`<feDropShadow>`。

-   **`dx`, `dy`**：偏移量。
-   **`stdDeviation`**：模糊半径。
-   **`flood-color`**：阴影颜色。

## 8.3.3 `<feColorMatrix>`：颜色矩阵

这是最强大的色彩调整工具。它允许你通过一个 4x5 的矩阵来操作 RGBA 通道。

最常用的快捷方式是 `type="saturate"`（饱和度）和 `type="hueRotate"`（色相旋转）。

```xml
<!-- 变为灰度 -->
<feColorMatrix type="saturate" values="0" />
```

## 8.3.4 `<feTurbulence>`：湍流噪声

这是 SVG 滤镜中最神奇的部分。它可以生成柏林噪声（Perlin Noise）。这种噪声看起来非常自然，常用于模拟云、水纹、大理石等自然纹理。

-   **`baseFrequency`**：噪声的频率（粗糙度）。值越小越平滑，值越大越嘈杂。
-   **`numOctaves`**：细节层数。层数越多，纹理越细腻，但计算越慢。

结合 `<feDisplacementMap>`（置换贴图），你可以用这个噪声来扭曲原始图像，产生水波纹或毛玻璃效果。

## 8.3.5 `<feMorphology>`：形态学操作

这听起来很学术，但实际上很简单：它可以让图形变胖或变瘦。

-   **`operator="dilate"`**（膨胀）：让亮色区域扩张。用于加粗文字。
-   **`operator="erode"`**（腐蚀）：让亮色区域收缩。用于变细文字。
